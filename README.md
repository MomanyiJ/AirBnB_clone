THE CONSOLE
===========

********
The AirBnB Clone Console is the genesis of the AirBnB project. In this part, we are going to create a commandline interpreter to manage the objects of the AirBnB Clone project.


###### The console should:

- Create a new object(ef: new User or ne Place)
- Retrieve an object from a file eg. a database
- Update an abject and it's attributes
- Perform operations on objects such as counting.
- Destroy an object
